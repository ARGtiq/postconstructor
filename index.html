Синхронизация

Ключи Supabase хранятся только в этом браузере/устройстве и не попадают в исходный код приложения. Взять их: Supabase → Settings → API.

После сохранения зайдите в «Проекты» → войдите по email, чтобы синхронизировать библиотеку между устройствами.
